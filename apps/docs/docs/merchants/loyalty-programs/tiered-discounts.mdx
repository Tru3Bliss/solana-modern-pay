# Tiered Discounts

This program allows merchants to setup a tiered discount program where customers are rewarded a discount based on their total spend. For the merchant, they select a threshold spend, a % discount, and a name for the tier. For the customer, once they have reached the threshold spend, their total is discounted by the % discount.
